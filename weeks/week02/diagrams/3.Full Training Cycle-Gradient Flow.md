# Full Training Cycle — Gradient Flow

🎯 **Focus:** Forward + backward + update

```mermaid
sequenceDiagram
participant User
participant Model
participant Layer
participant Ops
participant Backend
participant Autograd
participant Optimizer

User->>Model: forward(x)
Model->>Layer: compute
Layer->>Ops: forward ops
Ops->>Backend: execute
Backend-->>Model: y_pred

Model->>Autograd: compute loss
Autograd->>Backend: record ops
Autograd->>Layer: backprop
Layer->>Ops: gradient ops
Ops->>Backend: compute gradients
Backend-->>Autograd: grads

Autograd-->>Optimizer: gradients
Optimizer->>Backend: apply_gradients
Backend->>Model: update weights
```

## What this shows

- Full learning cycle
- Two directions: forward (prediction) and backward (learning)
- Optimizer as state-mutation component
